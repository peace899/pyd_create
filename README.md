# Rust Python Extension

A template for creating high-performance Python extensions using Rust and PyO3, compiled as `.pyd` (Windows)

## ✨ Features

- 🚀 **Native Performance** - Write performance-critical code in Rust
- 🐍 **Seamless Integration** - Import and use like any Python module
- 🔧 **Easy Setup** - Build with `maturin` or manual `cargo` configuration
- 📦 **Cross-Platform** - Works on Windows, Linux, and macOS

## 📋 Prerequisites

- [Rust](https://www.rust-lang.org/tools/install) (latest stable)
- [Python](https://www.python.org/) (3.7 or later)
- [pip](https://pip.pypa.io/en/stable/installation/)
